## Що це

**9.2** **IPSC PCC** визначає **методи підрахунку** для змагань **PCC** (назви, формули, коли який застосовується — лише **PDF Jan 2026**). **RO** фіксує **сирі дані** (час, попадання, штрафи, тип сцени за **WSB**); **скоринг** застосовує **9.2**. **C160** — **Handgun** — для трасс **PCC** звіряйте текст **PCC**.

## Що робить суддя змагань (RO)

1. **Звірити WSB** із **зареєстрованим** методом — без тихих замін.
2. **Заповнити** поля залікового листа, які вимагає **MD** / програма для дивізіонів **PCC**.
3. **Збої таймера / датчиків** — перестріл або виправлення за **MD**, не «ручна» математика.
4. **Формули** — зона відповідальності **статистики**; RO засвідчує **факти**.

## IPSC (Jan 2026)

- **9.2 PCC** + коефіцієнти дивізіону / додатків для події.
