## Суть процедури

Короткий розбір зміни: помилки **таймера**, команд **8.3.1**, інциденти **10.5**, уроки для **9.10** — у дусі **етики IROA** (конфіденційність, повага).

## Алгоритм дій RO

1. **Кейси:** DQ, зупинки, спірні залікові епізоди — факти, без імен у публічному полі.
2. **Безпека вправ:** Чи були «пастки» для **10.5**; що змінити на майбутніх етапах.
3. **Логістика:** **Таймер**, **відлуння**, мішені, хелпери — що гальмувало **9.10**.
4. **Етика IROA:** Конструктивний фідбек усередині бригади; рішення передавати **RM** / **MD** письмово для покращення процесів.

## Важливі нюанси (Jan 2026)

* Дебрифінг не замінює офіційний дисциплінарний канал **MD**.
* Критика колег — приватно; мета — безпечніший **обстріл** наступного матчу.

## Типові помилки RO

* Пропустити дебрифінг і повторити ті самі збої з **таймером** і відновленням етапу.
