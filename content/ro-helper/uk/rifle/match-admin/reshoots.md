## Що це

**4.7** описує, коли спортсмену дозволено **перестріл** через **несправність обладнання** майданчика — **без вини** спортсмена. Це **не** «другий шанс» після помилки в **COF** і не автоматичне право; **RM/MD** зазвичай фіксують процедуру та ліміти кількості перестрілів. **C300** — **Handgun**; для **4.7** потрібен **PDF Rifle**.

## Що робить суддя змагань (RO)

1. **Зафіксувати факт:** що саме зламалося; чи вплинуло на серію; чи однаково для всіх у ряду.
2. **Негайно повідомити RM/CRO**, не обіцяючи перестріл самостійно.
3. **Зберегти** заліковий лист / час до рішення, якщо інструкція матчу вимагає.
4. **Після дозволу RM** — провести перестріл за тими ж **WSB**-умовами, якщо PDF цього вимагає.
5. **Не використовувати** 4.7 як виправдання для **тренувального** проходу.

## IPSC (Jan 2026)

- Якір — **4.7 Rifle**; перехресно **9.10** (час), **C267** (несправність спорядження спортсмена — інша логіка).

## Типові помилки

- **RO оголошує перестріл** без **RM**.
- **Плутання** з перестрілом через **несправність зброї** (частіше **5**/інші пункти).
- **Різні правила** для «улюбленців» і решти ряду.
- Користуватися **Handgun 4.7** **без** відкритого **PDF Rifle**.
