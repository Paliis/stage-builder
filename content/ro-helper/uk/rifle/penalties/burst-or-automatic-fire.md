## Що це

**10.2.12** у **IPSC Rifle** стосується пострілу, коли **керування вогнем** або **характер випуску** (наприклад, **binary trigger**, **черга**, **повністю автоматичний** режим — **як визначено в PDF**) **не дозволений** для **дивізіону** спортсмена, правил обладнання або **COF**. Картка для **Rifle** — завжди відкривайте PDF **Rifle**. Для того самого номера правила в **PCC** / **Mini Rifle** див. **C102** / **C104**. Наслідки (**procedural**, **DQ** тощо) — **лише** з тексту **Jan 2026** та бюлетенів матчу / дивізіону.

## Що робить суддя змагань (RO)

1. **Підтвердити факти:** що саме спостерігалося (кілька набоїв на один навмисний натиск, неконтрольована черга, положення перемикача тощо) — **лише** за політикою доказів матчу та **безпечним** спостереженням.
2. **Читати 10.2.12** у **розділі 10 Rifle** разом із правилами обладнання **розділу 5** для дивізіону.
3. **Застосувати санкцію** суворо за PDF — не замінювати ярликами («звучить як binary») текст правила.
4. **Заліковий лист / RM:** якщо сумнів щодо класифікації чи легальності зброї — ескалація за процедурою матчу.
5. **Безпека:** за неконтрольованого вогню — спочатку **RM** і протокол майданчика, потім дискусія про залік.

- **Плутання дисциплін:** **shotgun** «неправильний набій» має інший контекст **10.2.12** — для **Rifle** використовуйте книгу **Rifle**.
