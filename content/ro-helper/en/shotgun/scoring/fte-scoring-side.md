## What this is

**FTE (failure to engage)** has **two layers** in the rules: **scoring** (**9.5.6**) and **procedural** (**10.2.7** — see card **C36**). This card is about **scoring**: how to record an un-engaged target, how many **misses**, interaction with **steel** / **paper**, etc. — **only** from the **Shotgun Jan 2026** PDF. For the **handgun** treatment of the same topic, see **C190**. The **RO** must not confuse “no hole” with other cases (wrong target type, uncredited knockdown).

## What the Range Officer (RO) does

1. **Confirm** the target **must** be shot per the **WSB** (not optional).
2. **Log** the format scoring requires: **miss** / **FTE** counts per **9.5.6 Shotgun**.
3. **Separately** enter **procedural** under **10.2.7** if the PDF requires both.
4. **If unclear** (target hidden by another, partial steel activation) — **CRO/RM** before finalising the sheet.
5. **Do not invent** “half an FTE” — only categories the rules allow.

## IPSC (Jan 2026)

- Read **9.5.6 Shotgun** **with** **10.2.7**; application order is per PDF text. Use **Shotgun Chapter 9**, not **Handgun** memory alone.

## Local (FPSU)

Software “FTE” fields must match match rules. **`fpsu_delta_verified`**: **false** until **RO_HELPER_V0 §4.1** review.

## Common mistakes

- **Procedural only** without score impact, or vice versa, when the PDF needs both.
- **Confusing** with a target that may be skipped per **WSB**.
- **Late edits** after signing without **9.6**.
- **Applying handgun-only examples** from **C190** when **Shotgun** wording differs.
