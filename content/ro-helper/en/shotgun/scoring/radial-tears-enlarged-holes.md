## What this is

**9.5.4** **IPSC Shotgun** addresses **irregular** holes and **tears** on **paper** or **card** targets used in **Shotgun** — pellet patterns differ from pistol holes; **9.5.4 Shotgun** + **appendices** define what counts. **C185** is **Handgun**; **C179** = **Shotgun** paper policy.

## What the Range Officer (RO) does

1. **Use** **Shotgun**-appropriate **gauges** / **MD** tools when **9.5.4** calls for them.
2. **Do not** import **handgun** overlay logic **without** checking **Shotgun** diagrams.
3. **Mixed** steel + paper stages: complete **paper** scoring **before** the squad leaves if **MD** requires preserved targets for **9.6**.

## IPSC (Jan 2026)

- **9.5.4 Shotgun** inside **9.5**; read with **C179**.

## Common mistakes

- **Treating** pellet **rips** like **9 mm** tears — wrong mechanics; use **Shotgun** rules.
- **No target retention** when **MD** mandates it for protests.
