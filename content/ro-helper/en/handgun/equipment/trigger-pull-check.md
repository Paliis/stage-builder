## What this is

**Handgun Jan 2026** sets **minimum trigger pull weight** for **Production** (and possibly other divisions) in **Appendix D** — the product matrix cites **Appendix D / D4 §6** and **2.27 kg** as a planning anchor. **You must confirm the exact number and wording in the official PDF** before treating any value as binding; other divisions use **different appendices** and limits.

## What the Range Officer (RO) does

1. **On the bay**, you normally **do not** run a certified pull test — that is **equipment inspection** / **RM** / **official scales** territory unless **MD** assigns it.
2. **If asked** whether a gun “passes”, answer: **only the authorised check** per **Appendix D** and match briefing counts.
3. **Hold the start** if you see an obvious mechanical safety issue (e.g. trigger moves with safety on) and call **RM** — separate from pull weight, but urgent.
4. **Do not improvise** with luggage scales or rubber bands; non-standard methods create protests.
5. **Escalate** disputes after an official measurement to **RM**, not social media.

## IPSC (Jan 2026)

- Read **Appendix D** (Production) in your **Handgun** PDF; verify **D4 §6** (or the current sub-clause if renumbered) for **pull direction**, **measurement point**, and **minimum force**.

## Common mistakes

- **Bay RO “certifying”** trigger weight verbally without authority.
- **Mixing** Production limits with **Open / Classic** configurations.
- **Quoting 2.27 kg** from memory when the **Jan 2026** appendix differs — always **open the PDF**.
