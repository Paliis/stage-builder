## What this is

**Chapter 6** of **IPSC Handgun** covers **match organisation**: stages (qualification, finals, etc. — per PDF), **teams**, **categories** by age/gender, progression rules. The bay **RO** rarely builds the schedule but should **know** whether the current string is in an allowed phase, how many **attempts** exist, and where to **send** questions on **class** or **team composition**.

## What the Range Officer (RO) does

1. **Know from MD briefing:** passes per stage, **shoot-off** rules, attempt caps.
2. **Do not allow** extra “practice runs” if the regs forbid them.
3. **Check** with the tablet / start officer that the competitor is in the **right category** for that block if the match splits squads.
4. **Notify RM** if the structure looks wrong (e.g. finals without the prior stage per rules).
5. **Keep parity** between squads — same attempts and same **WSB** conditions.

## Common mistakes

- **Bonus attempt** “because the timer glitched” without **RM** process.
- **Different attempt counts** across squads on the same stage without **WSB/MD** update.
- **RO settles** team roster disputes instead of **scoring / RM**.
