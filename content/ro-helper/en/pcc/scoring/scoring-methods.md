## What this is

**9.2** **IPSC PCC** defines **scoring methods** for **PCC** matches (names, formulas, when each applies — **Jan 2026** PDF only). The **RO** records **raw data** (time, hits, penalties, stage type per **WSB**); **scoring** applies **9.2**. **C160** = **Handgun** — confirm **PCC** text for PCC stages.

## What the Range Officer (RO) does

1. **Match WSB** to the **registered** scoring method — no silent switches.
2. **Complete** scoresheet fields required by **MD** / software for **PCC** divisions.
3. **Timer / sensor issues** — reshoot or fix per **MD**, not improvised math.
4. **Defer** formula debates to **stats** — RO certifies **facts**, not global points.

## IPSC (Jan 2026)

- **9.2 PCC** + division / appendix coefficients for the event.
