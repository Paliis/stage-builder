# Зворотний зв’язок користувачів (чернетка)

Сюди вставляйте повідомлення, скріншоти, цитати з чатів / листів. Файл для внутрішнього використання; не публікується на сайті.

**Версійність:** з **2026-04-06** відгуки ведемо в контексті продуктової лінії **V1** (усе після зміни розміру площадки включно). Межі V0 / V1 і накопичувальний опис релізів — у **[VERSIONING.md](./VERSIONING.md)**.

**Формат (за бажанням):** дата → джерело (Telegram, email, усно) → суть → статус (нове / в роботі / зроблено / відхилено).

---

## Вхідні записи

- **Ідеї (реквізит / мішені)** — щити з порталами (нижній, середній, високий); щити з портами з дверцями; косий порт; стілець і стіл з позначкою, стійка зі зброєю; Mini IPSC папір; металева мішень на середній стійці (50 см від пола); металева на високій стійці (1 м від пола).

- **Нове** — Треба додати функцію **копіювання одного об’єкта** та **групи об’єктів** (мішені / реквізит на плані): дублікат зі зміщенням або в буфер обміну — уточнити при реалізації.
