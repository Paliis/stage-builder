# Зворотний зв’язок користувачів (чернетка)

Сюди вставляйте повідомлення, скріншоти, цитати з чатів / листів. Файл для внутрішнього використання; не публікується на сайті.

**Версійність:** з **2026-04-06** відгуки ведемо в контексті продуктової лінії **V1** (усе після зміни розміру площадки включно). Межі V0 / V1 і накопичувальний опис релізів — у **[VERSIONING.md](./VERSIONING.md)**. Ідеї для наступної великої хвилі можливостей — у розділі **«Накопичення для V2»** нижче.

**Формат (за бажанням):** дата → джерело (Telegram, email, усно) → суть → статус (нове / в роботі / зроблено / відхилено).

---

## Вхідні записи

- **Ідеї (реквізит / мішені)** — стілець і стіл з позначкою, стійка зі зброєю. *(Щити: низький/високий/косий порт і дверцята в порті — у редакторі як окремі типи реквізиту.)* *(Мішені: Mini IPSC; метал на стійці 50 см / 1 м.)*

- **Нове** — Треба додати функцію **копіювання одного об’єкта** та **групи об’єктів** (мішені / реквізит на плані): дублікат зі зміщенням або в буфер обміну — уточнити при реалізації.
